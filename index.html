Screen Width Comparisons

If your screen is less than 1920px in width, this page will cause horizontal scrolling.

Desktop, Mobile and Tablet sections have bars that show the width of the screen in pixels, along with the percentage of US devices that have that width in 2020. The order is from most-frequent to least-frequent. The actual physical size of the bars on the device's screen will vary based on pixel density. The resolution in each bar shows the screen width first and the height second.

Bootstrap small, medium & large sizes are shown. Extra-small is not shown, but is less than small. Extra-large is not shown, but is greater than large.
